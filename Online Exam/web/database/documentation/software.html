- Software -

jMailComposer is a 100% JAVA JSP/Servlets upload component. Here are the three requirements you need to run it :

• Servlet Engine or Application Server compliant JSP1.1/Servlets2.2 API (or higher).
• JVM 1.2 or higher.
• Browser (Internet Explorer 4.x or higher, Netscape 4.x or higher)
• Optionally a JavaMail and JAF implementation.

jMailComposer should run under ANY platform supporting JSP1.1/Servlets 2.2 or J2EE 1.2 (or higher).
